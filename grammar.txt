<PROGRAM>                      -> <PROGRAM TAIL> | ε
<PROGRAM TAIL>                 -> <FUNCTION DEFINITION> | <STATEMENT LIST> | ε

<FUNCTION DEFINITION>          -> Define function <FUNCTION DECLARATION>:<STATEMENT LIST>End function.
<FUNCTION DECLARATION>         -> <WORD LIST>(PARAMETER)<FUNCTION DECLARATION TAIL>
<FUNCTION DECLARATION TAIL>    -> <FUNCTION DECLARATION> | ε

<STATEMENT LIST>               -> <STATEMENT><STATEMENT LIST TAIL>
<STATEMENT LIST TAIL>          -> <STATEMENT LIST> | ε
<STATEMENT>                    -> <STATEMENT PORTION><STATEMENT TAIL>.
<STATEMENT TAIL>               -> <STATEMENT PORTION> | ε
<STATEMENT PORTION>            -> <WORD LIST><PARAMETER>

<WORD LIST>                    -> <WORD><WORD LIST TAIL>
<WORD LIST TAIL>               -> <WORD LIST> | ε

<PARAMETER>                    -> <VARIABLE> | <EXPRESSION> | <STRING> | <NUMBER> | <BOOLEAN>
<VARIABLE>                     -> <WORD LIST>
<EXPRESSION>                   -> <PARAMETER><OPERATOR><PARAMETER> | (<PARAMETER>)

<WORD>                         -> <string of letters only>
<STRING>                       -> "<any characters, backslash escapes special characters>"
<NUMBER>                       -> <standard numbers; integers and decimals>
<BOOLEAN>                      -> true | false
<OPERATOR>                     -> + | - | * | / | ^
